HW15

Написать консольную игру "камень-ножницы-бумага":

1.Пользователь должен выбирать количество игр и ввести свое имя.
2.Пользователь должен иметь возможность прервать игру.
3.После прекращения игры пользователь должен увидеть результат: сыграно игр, выиграно, проиграно.
4.Функционал игры должен быть покрыт тестами - надо написать тест на метод который будет определять победителя.
5.Игру упаковать в .jar file и приложить его к ДЗ в LMS + линка на исходный код.
